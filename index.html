Simple Sculpt Layers Plus

Professional Non-Destructive Sculpt Layer System for Blender

Version 1.0 Blender 4.5+
📖 Overview: Simple Sculpt Layers Plus (SSLP) emulates a professional sculpt layer system by storing vertex offsets relative to a true base mesh. Provides non-destructive sculpting workflows similar to ZBrush, with support for multi-resolution sculpting, quick poses storage, and shape key management.

📦 1. Installation

  1. Download the addon (SSLP.zip)
  2. In Blender, go to Edit → Preferences → Add-ons
  3. Click Install from Disk and select the SSLP.zip file
  4. Enable the addon by checking the box next to Simple Sculpt Layers Plus
  5. The panel will appear in the Viewport Side Panel (N) → My Tab

    6. ℹ️Note: You can change in which tab the addon panel would be, inside the addon preferences

🧱 2. Multi-layer

Overview: The multilayer system allows you to stack sculpting passes non-destructively. Each layer stores vertex offsets relative to the base mesh.

Set Base Mesh

Stores the current mesh state as the foundation for all sculpt layers. Every mesh is required to have a base mesh before user can start working with the Multi-Layer panel

Add Sculpt Layer

Creates a new empty layer at the top of the stack, and enables it

ℹ️ Click on the new layer name to rename it

Enable Layer

Sets the active layer for storing/erasing sculpt changes (offset data)

ℹ️ Only one layer can be enabled at a time

Delete Sculpt Layer

Removes the selected layer and its associated vertex offset data

Move Sculpt Layer Up/Down

Reorders layers in the stack. Use when you want to merge down a two specific layers.

Merge Down

Merges the enabled layer with the one below

Toggle Visibility

Shows or hides the layer in the viewport

ℹ️ Hidden layers cannot be enabled

Set Layer Strength

Controls how much the layer contributes to the overall mesh (1-100%)

Cut Mask to New Layer

Extracts the layer offsets of the enabled layer - that lie under the masked area - into a new layer

ℹ️ Sensitive to mask strength

Remesh

Uses Blender's remesh operator in a way that is compatible with Mult-Layer

ℹ️ You can assign this button to a hotkey, and even replace the default 'Remesh' keyboard shortcut

Eraser Brush

Creates and activates a specialized 'LAYER' type brush that deletes only the offsets of the enabled layer

Mask Erase

Much like the 'Eraser Brush' - Removes any offsets of an enabled layer, that lies within a masked area. Use when you need a precised & controlled deletion

ℹ️ Sensitive to mask strength

Multi-res Operations

Create Multi-res

Creates a corresponding multi-res mesh to work with your multi-layer mesh. Only available when no corresponding multi-res mesh is found

Convert to Multi-res

Converts your multi-layer mesh to its multi-res counterpart, using the 'Reshape' function of the Multires modifier

Rebuild from Retopology

Rebuilds the multi-layer using a retopology mesh /w shrinkwrap modifier. Useful for when 'Create Multi-res' gives you a multi-res mesh with 0 levels, due to the multi-layer having bad topology:

Example of bad topology that causes Create Multi-res to fail

For best results follow these steps:

    Step 1: Once you have create a retopology mesh, make sure it has 2 modifiers: Multires & Shrinkwrap (in that order). Subdivide the multires as needed in order for the shrinkwrap to capture all the details of you multi-layer

    Step 2: Make sure each layer of the multi-layer is shrinkwrapped properly by the retopology mesh. Then, simply click the 'Rebuild From Retopology' button and select your retopology mesh.

    You will now have a multi-layer + multi-res setup & ready to go. the old multilayer will be hidden and given "_old" prefix to its name.

🔍 3. Multi-res

Reshape method:

The 'Reshape' method keeps copies of the multires and recalls them using the 'Reshape' function of the multires modifier. Use when you would like to save different iterations made on your multires mesh, whenever the multires level is > 0

Add Pose

Creates a new empty pose slot

ℹ️ Click on the new pose name to rename it.

Delete Pose

Removes the selected pose and its data

Clear to Basemesh

Resets the mesh to its base state, removing all pose deformations

Store Pose

Saves the current mesh deformation as a pose

Apply Pose

Applies a stored pose to the current mesh

Bake to Multi-Layer

Bakes the current pose into a sculpt layer.

ℹ️ In addition, this reloads the layer status that the multi-layer had, when the pose was first stored. This is done so that the newly baked layer, would not re-store the offsets that are already stored in previous layers

Bake to Shapekey Mesh

Bakes the current pose to a shape key on the mesh

Shape Keys method:

The Shape Keys method uses Blender's native shape key system for pose storage. Use when you would like to save different iterations made on your multires mesh, whenever the multires level is = 0

Add Shape Key

Creates a new shape key & sets the multires level to 0.

ℹ️ Click on the new shapekey name to rename it.

Delete Shape Key

Removes the selected shape key

Toggle Shape Key Mute

Mutes or unmutes the selected shape key.

Restore Shape Keys

Restores all shape keys from the the stored '_Shapekeys##' mesh. If '_Shapekeys##' mesh is present, you can use this function when you would like to reload your shapekeys.

* '_Shapekeys##' mesh - is created automatically when 'Delete Lower' or 'Convert to Multilayer' (Case 3) functions are called. This mesh keeps and restore the user saved shape keys, which would otherwise get deleted in the process

Clear All ShapeKeys

Removes all shape keys from the mesh & deletes the stored '_Shapekeys##' mesh. Use if the stored '_Shapekeys##' gets corrupted

Delete Lower

Deletes the lowest subdivision level of the multires mesh, while preserving shape keys & poses. Use when the multires mesh base geometry is not enough to make shapekey poses

⚠️ Note: Depending on the size of the mesh, this process might take a while. Open the console beforehand if you would like to track the progress (Window >> Toggle System Console)

Multi-Layer Operations

Create Multi-Layer

Creates a corresponding a multi-layer setup for the current mesh. Only available when mesh has a multires modifier & no corresponding multi-layer

Convert to Multi-Layer

Switches back to Multi-Layer. Only available when mesh has a multires modifier & a corresponding multi-layer

This function runs under 3 different cases:

Case 1:

Multi-res level = Multi-Layer level - In which case the conversion will be fast and will not take more then a few seconds

Case 2:

Multi-res level < Multi-Layer level - slightly slower then case 1, as the conversion will subdivide the multi-res until it matches the multi-layer level

Case 3:

Multi-res level > Multi-Layer level - the slowest case. Here the Multi-Layer would be reconstructed from the ground up with the appropriate subdivision levels, to properly match the indices order of the multi-res

⚠️ Note: Depending on the size of the mesh, this process might take a while. Open the console beforehand if you would like to track the progress (Window >> Toggle System Console)
